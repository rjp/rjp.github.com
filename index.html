Creations

  • Find and describe links http://github.com/rjp/linkylinky
    linkylinky finds URLs in text and, with the help of plugins, describes them.
  • URL catching Jabber MUC bot http://github.com/rjp/ribot
    The ribot bot sits on a Jabber MUC and uses the linkylinky library to fetch information about the URLs it sees. The default behaviour is to fetch the initial 16k of the URL and look for a <title> tag. Plugins are provided which return more useful information - for example, the twitter plugin returns the text of the tweet.
  • Self-referential album art http://rjp.github.com/cdcover/ [Source]
    Shell scripts and ruby code to generate album or track art from the music's own waveform.
  • Socket to Jabber bridge http://github.com/rjp/jabber_cat
    Inspired by irc_cat but with the output being a Jabber MUC rather than an IRC channel.
  • IRC to Jabber bridge http://github.com/rjp/irc_muc
    Semi-inspired by ejabberd_ircd, irc_muc lets a user talk to a Jabber MUC using their irc client of choice.
  • CDmeme parts generator http://cdmeme.trakr.nu/
    Simple Sinatra app that utilises the flickr API and some screen-scraping to randomly select the parts for your album cover and remembers them in a database for later retrieval.
  • Fake album covers etc. http://fakestuff.org/
    Tumblr with both cdmeme- and none-cdmeme inspired fake album covers (and the occasional fake book.)
